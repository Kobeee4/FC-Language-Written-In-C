# FCL - FuseCore Language
# THINGS CAN UPDATE SO THE WORDS WILL BE INACCURATE ONCE I RELEASE IT
> **Status:** 🚧 Coming Soon

FCL (FuseCore Language) is a new coding language designed for performance, simplicity, and seamless system integration. Built for modern development workflows, FCL aims to bridge the gap between low-level control and high-level expressiveness.

---

## 📁 File Format

| Extension | Description |
|-----------|-------------|
| `.fc`     | FuseCore Language source file |

---

## 🚀 Usage

Once installed, FCL can be invoked via the `fcl` command:

```bash
# Run an FCL source file
fcl filename.fc
```
